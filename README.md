# HomeDNS Analytics

> A lightweight, self-hosted DNS filtering and network analytics platform built with Go, FastAPI, React, and SQLite.

HomeDNS Analytics is a personal engineering project focused on building a production-ready DNS filtering appliance for home networks.

The project aims to provide:

- 🚫 DNS-based ad and tracker blocking
- 📊 Real-time and historical DNS analytics
- ⚡ High-performance Go DNS resolver
- 🔒 Secure FastAPI management API
- 💻 Responsive React dashboard
- 🍓 Optimized for Raspberry Pi 3B
- 🔄 CI/CD with GitHub Actions
- 🧪 Automated testing across all components

## Tech Stack

| Component      | Technology     |
| -------------- | -------------- |
| DNS Service    | Go             |
| Management API | FastAPI        |
| Frontend       | React + Vite   |
| Database       | SQLite         |
| Deployment     | systemd        |
| CI/CD          | GitHub Actions |
